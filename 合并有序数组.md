# 合并有序数组
####**实验目的：合并有序数组**
####实验内容：给定两个升序数组，合成一个输出 

判断题目要求

##Input
~~~~~
-第一行第一个有序数组长度正整数n，n<=20；
-第二行第一个有序数组使用n个数字，使用空格隔开
-第三行第二个有序数组长度正整数m，m<=20;
-第四行第二个有序数组包含m个数字，使用空格隔开。
~~~~~
---------------
##Output
~~~~~
输出合并后的数组，每个数字占一行
~~~~~
-------
### **实现这个功能的方式有如下几种**

- ##方法1
~~~~~
-新建数组c，该数组的大小>=已知两个数组之和
-比较两个数组中的元素
-将较小者放到空数组中，直到其中一个数组为空
-最后把剩下的数组全部迁移新创建的数组中
~~~~~
- ## 方法2
~~~~~
若某个较大的数组a的末尾有足够的空间容纳数组b，将数组b容纳到数组a中
~~~~~
--------------

#include <stdio.h>
#include <stdlib.h>
 
typedef struct Node
{
    int data;
    struct Node *next;
}LNode,*p;
 
void debugprint(LNode *p)
{
    return ;
    for(; p; p=p->next) printf("<%p: %d,%p> ", p, p->data, p->next);
    puts("");
}
 
LNode* CreateLinklist(int n)
{
    LNode *head,*p,*q;
    int i;
    head=(LNode*)malloc(sizeof(LNode));
    head->next=NULL;
    p=head;
    q=p;//指针q始终指向链尾节点
    for(i=1;i<=n;i++)
    {
        p=(LNode*)malloc(sizeof(LNode));
        scanf("%d",&p->data);
        p->next=NULL;
        q->next=p;
        q=p;
    }
    return head;
}
 
LNode * MergeLinklist(LNode* a,LNode* b,int m,int n)
{
    LNode* k=a;
    LNode* p=a->next;
    LNode* q=b->next;
    LNode* C=k;
    while(p&&q)
    {
        if(p->data<=q->data)
        {
            k->next=p;
            k->next->data=p->data;
            p=p->next;
            k=k->next;
        }
        else
        {
            k->next=q;
            k->next->data=q->data;
            q=q->next;
            k=k->next;
        }
    }
    k->next=p?p:q;
    return C;
}
 
void print(LNode *p)
{
    LNode *q;
    q=p->next;
    while(q!=NULL)
    {
        printf("%d\n",q->data);
        q=q->next;
    }
}
 
int main()
{
    int m,n;
    LNode *A,*B,*D;
    scanf("%d",&m);
    A=CreateLinklist(m);
    scanf("%d",&n);
    B=CreateLinklist(n);
    debugprint(A);
    debugprint(B);
    D=MergeLinklist(A,B,m,n);
    debugprint(D);
    print(D);
    return 0;
}
